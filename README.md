# Wordle Clone

A simple Wordle game built with React.

## Features
- Dynamic guesses
- Remaining attempts counter
- Win / Lose states
- Reset game
- Keyboard support (Enter key)
- Random word API (with fallback)
- Error handling
- Used letters tracker
- Dark mode UI
- Controlled input

## Tech
- React
- useState
- useEffect
- API
- Conditional Rendering
